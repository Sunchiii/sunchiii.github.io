## Getting Started

Requires Node.js v22.x and pnpm.

```sh
git clone https://github.com/Sunchiii/potfolio2.0
pnpm install
pnpm dev        # http://localhost:4321
pnpm build      # type-check + build
pnpm preview    # preview production build
```
